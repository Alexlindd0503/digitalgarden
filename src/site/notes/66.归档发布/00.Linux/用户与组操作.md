#linux

```ad-summary
title: 总结

- useradd/userdel 管用户，groupadd/groupdel 管组，usermod 改用户所属组
- wheel 组是管理员组，加进去就有 sudo 权限
```

## 1. 用户操作

```bash
useradd 用户名                    # 添加用户
useradd -g 组名 用户名             # 添加用户并指定所属组
useradd -d /home/ldd ldd          # 添加用户并指定主目录
passwd 用户名                     # 修改密码
id 用户                           # 查看用户信息
cat /etc/passwd                   # 查看所有用户
su 用户                           # 切换用户
sudo xx                           # 临时 root 权限执行命令
vim /etc/sudoers                  # 配置 sudo 权限
userdel 用户                      # 删除用户
```

## 2. 组操作

```bash
groupadd 组名                     # 添加组
cat /etc/group                    # 查看所有组
usermod -g 组名 用户名             # 修改用户所属组
groupmod -n new old               # 组改名
groupdel 组名                     # 删除组
```

`wheel` 组是管理员组，把用户加进去就能用 sudo。
