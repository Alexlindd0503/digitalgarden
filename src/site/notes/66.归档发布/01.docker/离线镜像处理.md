#docker

```ad-summary
title: 总结

- 内网没法 pull 镜像，可以在联网机器 `docker save` 导出 tar，传进去再 `docker load` 导入
```

## 1. 导出镜像（联网机器）

```bash
docker save java:8 -o java.tar

# 压缩方便传输
gzip java.tar
```

## 2. 导入镜像（内网机器）

```bash
gunzip java.tar.gz

docker load -i java.tar
```
