### 1. 用户操作
```bash 
useradd 用户名 #添加用户
useradd -g 组名 用户名  #添加新用户到某个组
useradd -d /home/ldd ldd  #添加用户ldd 给home/dd为主文件夹目录
passwd 用户名  #更改用户密码
id 用户  #查看用户信息
cat /etc/passwd  #查看所有用户信息
su 用户  #切换用户
sudo xx  #给用户临时root权限
vim /etc/sudoers #进行权限赋予
userdel 用户  #删除用户
```

### 2. 组操作

```bash
groupadd 组名  #添加组
cat /etc/group #查询所有组
usermod -g 组名 #修改用户组
groupmod -n new old  #组改名
groupdel 组名  #删除组
```
